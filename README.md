📊 CNBC Clone Website

A responsive CNBC News Website Clone built using only HTML & CSS.
This project focuses on replicating the layout, design, and user interface of the CNBC website.

🚀 Features

📰 News Homepage Layout

📊 Business & Market Sections UI

🎨 Clean and Modern Design

📱 Fully Responsive (Mobile + Desktop)

⚡ Fast Loading (No JavaScript)

🛠️ Tech Stack

HTML5
CSS3 (Flexbox & Grid)
